## MealHunt
### mealhunt is a flutter app with 
> Recipes from around the world 
according to [TheMealDB.com](https://www.themealdb.com/). 
It uses their free JSON API to 
- browse through different categories
- search a meal
- surprise me! for a random meal

### Used api endpoints are inside "lib/constants/api_endpoints.dart"

Thanks for the API! :+1:
